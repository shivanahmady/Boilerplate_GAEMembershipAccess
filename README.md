# Admin-Portal-Access-App
Sample app of user members login/logout/admin access on Google Cloud App Engine. (WebApp2+Jinja2)


Prerequisites: 
----------------
- Gcloud, Webapp2, Jinja2


To Start:
-----------------
- `dev_appserver.py app.yaml`

To Access:
-----------------
- localhost:8080
